Creating a Desktop application with the Electron.js

Prerequisites:

1. Node.js
2. MySQL


PART 01 - Installation, Configure and Project Understanding

First we go use the Quick Start for create a aplication quick of Electron. This aplication provide an initial struture for the project.

Clone the repository Quick Start:
$ git clone https://github.com/electron/electron-quick-start

In the repository:
$ cd electron-quick-start

Install the dependencies and execute for the first time the aplication:
$ npm install && npm start

The install dependencies will go be equal or similar to image below:


The execution of the aplication will be similar to the image below:


If you look, some files were downloaded in the project, among them are main.js, index.html, package.json and the folder node_modules. We’ll to talk a little about each of them now.

main.js

This file is set up for us with most of the resources needed for the start of a project. Some of these settings are the size of the application window, the path to the first screen, among other things.

index.html

This file is the page or the screen application early (since we are talking about a desktop application). Here you can create a design and some initial interaction to other screens using features such as CSS, Bootstrap, among others, as creating a Web application.

package.json

All necessary dependencies that arise during the development of your project you can install by running npm install. For example, suppose you go to work with MySQL, then you come to this file and in “devDependencies” adds "mysql": "*", with this the latest version of MySQL is added in your folder node_modules and is available to be used by you in the project using the command require(“mysql”);.

node_modules

In this folder are installed all the dependencies of the project, such as MySQL how was spoken in the previous topic.
